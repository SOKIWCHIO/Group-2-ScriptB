Software development for probabilistic modeling

11 个月

A useful #LLM trick for software development:
- Generate local coverage
- Show the `lcov.info` file to the model
- Ask it to generate tests to improve coverage
- Iterate as needed

At each step, tests either pass (and you get some candidate tests) or fail (and you find bugs in either the code or the tests).

Also important to ask it not to generate magic numbers. Any number in tests should be arbitrary or have a clear source.
