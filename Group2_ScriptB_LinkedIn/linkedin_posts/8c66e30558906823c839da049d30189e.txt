Multiple Tools: Uses various algorithms like SVMs (Support Vector Machines) and Clustering techniques.
Traditional Approach: In ML, the process typically involves manual feature extraction.
Process Flow: INPUT → Feature Extraction → Classification → OUTPUT
Data Requirements: ML models can perform well with smaller datasets.

Modern Approach: DL automates feature extraction and classification.
Process Flow: INPUT → Feature Extraction + Classification → OUTPUT
